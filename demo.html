jQuery repeatedclick plugin

This plugin allows developers to bind a function to a repeatedclick event which occurs when the mouse button is held. Like all jQuery functions, it is bound for all matching elements.

Repeatedclick event is fired when the pointing device button is clicked over an element. A repeatedclick is defined as a sustained mousedown event over the same screen location. The sequence of these events is:

  • mousedown
  • repeatedclick
